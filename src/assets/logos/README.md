# Logos Directory

This directory contains logo assets for the Excelr8 application.

## Current Logo

The Excelr8 logo is implemented as a React component in `src/components/Excelr8Logo.jsx`.

## Logo Design

The logo features:
- A stylized "8"/"B" shape
- Small rounded dot on the left
- Short rounded horizontal bar connecting to the main shape
- Two stacked horizontal bars (top shorter, bottom longer)
- Vertical rounded segment on the right connecting them
- Closed loop at the top

## Usage

The logo component automatically adapts to the current theme:
- Black color for light themes (Excelr8 2.0, Upwork)
- White color for dark themes (Default, Excelr8, Test Theme1)
